# Contributing

Thanks for helping improve AgentRadar.

## Development

```bash
swift build
swift run AgentRadar
swift test
```

Use the existing SwiftPM layout and keep changes focused.

## Pull requests

- Describe the user-visible change.
- Keep generated build outputs out of commits.
- Avoid new dependencies unless the benefit is clear.
- Update README files when behavior or setup changes.

## App icon

The icon source is generated by:

```bash
./DesignAssets/make_app_icon.sh
```

Commit `Assets/AppIcon.icns` and `Assets/AppIcon-1024.png` when the icon changes.
